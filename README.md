# Kuya Concierge

Scaffold for the Big Bros White Sand Resort voice concierge project.

## Structure

- `backend/`: FastAPI webhooks for ElevenLabs server tools
- `frontend/`: Guest-facing React app with reactive client tool UI
- `owner/`: Owner mode React app
- `knowledge/`: Knowledge base text files for ElevenLabs

## Status

This repository is scaffolded with minimal stubs only. Implement module-by-module.
